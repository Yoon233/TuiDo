# TuiDo

A Python ToDo List application which utilized TUI(Terminal User Interface). 

## Screenshots

![App Screenshot](https://via.placeholder.com/468x300?text=App+Screenshot+Here)

## Installation

Install my-project with git

```bash
git clone https://github.com/Yoon233/TuiDo.git
```
If you want to make it executable in shell:

## For linux 
```bash
### For the current user only:
cd TuiDo
cp tuido ~/.local/bin/ 

### If you want to make it available in all users:
cd TuiDo
cp tuido /etc/bin/
```
## For Windows


## Usage/Examples

```bash
tuido
```

## FAQ

#### Where do the todo lists are stored by default? 

For default the todo lists will be stored in ~/TuiDo
